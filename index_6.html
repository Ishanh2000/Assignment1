Resizable Modifiable Window Subwindow

Description
The following is a resizable (using mouse) and browser-modifiable iframe with source Wikipedia Home Page.
Try to open this at night.  
Note:
This page opens up the iframe below in default size of 200 px (height) X 300 px (width), independent of the browser window size. If the user drags the iframe lower right corner using mouse, the iframe changes size.
Also the URL can be entered independently!
