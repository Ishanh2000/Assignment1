Modifiable Adaptable Subwindow

Description
The following is a modifiable adaptable iframe with source Wikipedia Home Page. Hence its source and size both can be varied... try it out!
Try to open this at night.  
Note:
This page opens up the iframe below in default size of 200 px (height) X 300 px (width), independent of the browser window size. If the browser window is enlarged, the frame will enlarge, and vice-versa. So bigger size of frame than this is possible.
One can enter the site (full address preferred).
