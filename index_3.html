Modifiable Adaptable Subwindow

Description
The following is a modifiable adaptable iframe with source Wikipedia Home Page. Its source and size both can be varied... try yourself!
Try to open this at night. The background image is different for day and night.  
Note:
This page opens up the iframe below in default size of 200 px (height) X 300 px (width), independent of the browser window size. If the browser window is enlarged, the frame will enlarge, and vice-versa. So bigger size of frame than this is possible. One can choose from a list of sites available.
